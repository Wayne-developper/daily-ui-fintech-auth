# Daily UI — Dolara Fintech UI

A responsive fintech interface created for the Daily UI Day 001 challenge.

The project presents **Dolara**, a fictional financial brand, through two connected interface concepts:

- A promotional landing hero
- A responsive authentication screen

## Features

- Responsive desktop and mobile layouts
- Mobile-first content ordering
- Fintech-inspired visual identity
- Reusable CSS color variables
- Responsive Bootstrap grid
- Custom form controls
- Password visibility toggle
- Google and Facebook sign-in button concepts
- Accessible labels and semantic HTML
